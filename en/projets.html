C Compiler

End 2024

First-year beginning project in a group of 4. We implemented a C compiler to x86-64 assembly code. OCaml libraries are used to lex and parse the code to be compiled and create a JSON file. A Python file then interprets this JSON file to write an assembly file that behaves as intended by the original code. Pointers, local and global variables, arithmetic operations, arrays, functions, printf/scanf, if/else, while loops, and malloc are supported.

Repository link
